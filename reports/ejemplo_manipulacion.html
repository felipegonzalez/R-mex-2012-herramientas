Creaci√≥n de variables

Con la librer√≠a stringr se manipulan los c√≥digos para separarlos en las tres variables que interesan. 

library(stringr)
codigos <- unique(as.character(dat.m$variable))
  • str_split_fixed

Separa una cadena en distintas partes siguiendo un patr√≥n, en este caso (“_”), y devuelve las nuevas cadenas en una matriz que adem√°s se puede definir la dimensi√≥n

Sin embargo, en la primera columna todav√≠a se necesita extraer subcadenas con un patr√≥n determinado y obtener las dos varibles faltantes que dependen de las posiciones de los caracteres.

  • str_sub (equivale a strsub)

Extrae una cadena de un vector siguiendo un patr√≥n indicando las posiciones de inicio y final. Difiere en que regresa un vector de longitud cero si no existe el patr√≥n y se puede usar signo negativo para indicar que las posiciones se invierten. 

estacion <- str_sub(codigos.sep[, 1], start = -3, end = -1)

Como ninguna cadena del vector tiene m√°s de 10 posiciones se indica este y tomar las posiciones restantes y en caso de no haber simplemente lo omite 

medicion <- str_sub(codigos.sep[, 1], start = -10, end = -4)

Formato fechas

Por ultimo se preparan las fechas y se convierten en formato “POSIXlt” and “POSIXct” y en esta nueva fecha se agrega la hora del d√≠a en la que fue tomada con la funcion strptime.
Una vez en este formato es muy sencillo extraer el a√±o, mes, d√≠a y hora con la librer√≠a lubridate que tienen herramientas para manipular fechas. 

library(lubridate)
dat.j$fecha.hora <- strptime(paste(dat.j$FECHA, dat.j$HORA - 1), format = "%d/%m/%Y %H")
dat.j$a√±o <- year(dat.j$fecha.hora)
dat.j$mes <- month(dat.j$fecha.hora, label = TRUE)
dat.j$dia <- day(dat.j$fecha.hora)
dat.j$hora <- hour(dat.j$fecha.hora)
dat.j$FECHA <- NULL
